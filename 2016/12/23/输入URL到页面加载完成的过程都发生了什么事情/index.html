输入URL到页面加载完成的过程都发生了什么事情

发表于 2016-12-23 | 分类于 Frontend

1.输入url,按下回车后，这时你的浏览器首先查询DNS服务器，将url转换成ip地址。
2.DNS查询之后，返回IP地址给浏览器。
3.有了 IP 地址，就可以通过 Socket API 来发送数据了，这时可以选择 TCP 或 UDP 协议。
4.浏览器给web服务器发送一个HTTP请求。
5.服务器的永久重定向响应（从 http://example.com 到 http://www.example.com）。
6.浏览器跟踪重定向地址。
7.服务器处理请求。
8.服务器返回一个 HTTP 响应。
9.浏览器显示 HTML。
10.浏览器发送请求获取嵌入在 HTML 中的资源（如图片、音频、视频、CSS、JS等等）。
11.将渲染好的页面图像显示出来，并开始响应用户的操作。
12.浏览器发送异步请求。
